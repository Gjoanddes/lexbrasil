⚙️ Painel do Administrador

Acesso restrito. Ao buscar novidades, todos os visitantes verão as atualizações publicadas.
Como funciona: informe sua senha e API key, clique em buscar. A IA pesquisa na web as últimas novidades do STF, STJ, TST e Planalto e salva no banco de dados. Visitantes veem automaticamente em 🔔 Novidades.
SQL para criar tabelas no Supabase (rodar uma vez): CREATE TABLE novidades ( id SERIAL PRIMARY KEY, tipo TEXT, fonte TEXT, titulo TEXT, descricao TEXT, impacto TEXT, data_aproximada TEXT, criado_em TIMESTAMPTZ DEFAULT NOW() ); CREATE TABLE meta ( chave TEXT PRIMARY KEY, valor TEXT ); Variáveis no Vercel:
SUPABASE_URL · SUPABASE_ANON_KEY · SUPABASE_SERVICE_KEY · ADMIN_SENHA
